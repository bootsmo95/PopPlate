## Spacing Scale

Standard 8-point scale already used project-wide. Tailwind default scale applies; custom extensions listed below.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px (`gap-1`) | Icon gaps, inline padding |
| sm | 8px (`gap-2`, `p-2`) | Compact element spacing, badge padding |
| md | 16px (`gap-4`, `p-4`) | Default element spacing, card inner gap |
| lg | 24px (`gap-6`, `p-6`) | Card padding (`.p-card`), section padding |
| xl | 32px (`gap-8`) | Layout column gaps (`.two-col`) |
| 2xl | 48px (`py-12`) | Empty-state vertical padding |
| 3xl | 64px | Page-level vertical rhythm |

Exceptions:
- Stat cards on mobile: `p-[18px]` (18px — existing `.p-card` mobile override, source: main.css)
- Table cells: `px-5 py-3.5` (20px / 14px — existing `.p-table` pattern, source: DishTable.vue)
- Segmented tab controls: `px-4 py-2` (16px / 8px — standard filter-pill sizing)

**Source:** tailwind.config.ts extended spacing, main.css `.p-card` and `.p-table` styles

## Typography

All sizes and weights match the existing platform convention extracted from main.css, StatCard.vue, and DishTable.vue.

| Role | Font | Size | Weight | Line Height | Letter Spacing |
|------|------|------|--------|-------------|----------------|
| Body | Manrope (`font-body`) | 15px | 400 (regular) | 1.5 | normal |
| Label / Mono caption | JetBrains Mono (`font-mono`) | 11px | 500 (medium) | 1 | 0.18em (`.mono-label`) |
| Heading (card titles) | Cormorant (`font-display`) | 22px | 400 (normal) | 1.2 | -0.015em |
| Stat value (large number) | Manrope (`font-body`) | 48px (desktop) / 36px (mobile) | 300 (light) | 1 | -0.04em |

**Two weights only:** 400 (body/heading/display) and 500 (mono labels/buttons). Weight 300 is used exclusively on stat value numerals (existing StatCard pattern — do not apply elsewhere).

**Source:** StatCard.vue, main.css `.page-title` / `.mono-label`, DishTable.vue `.p-table th`

## Color

Warm linen palette. Already fully defined in tailwind.config.ts — do not introduce new color values.

| Role | Token | Hex | Usage |
|------|-------|-----|-------|
| Dominant (60%) | `bg` | `#f3ede2` | Page background, body surface |
| Secondary (30%) | `paper` / `card` | `#f8f3e9` / `#ede4d2` | Card surfaces (`.p-card`), table header, sidebar |
| Accent (10%) | `clay` | `#b87a4e` | Reserved for: focus rings, delta-up indicator, active nav item, CTA button hover, trend line chart stroke |
| Trend delta — positive | `#4a6240` | (inline) | Delta "up" text on stat cards — existing StatCard pattern |
| Trend delta — negative | `#8a4838` | (inline) | Delta "down" text on stat cards — existing StatCard pattern |
| Destructive | `status.failed` | `#a85a48` | Error states only — no destructive actions in this phase |
| Borders | `line` | `rgba(26,20,16,0.10)` | All card and table borders |
| Ink (text) | `ink.DEFAULT` | `#1a1410` | Primary text |
| Ink muted | `ink.mute` | `#6f5d4d` | Secondary / caption text |
| Ink faint | `ink.faint` | `#9a8773` | Timestamps, placeholder text, empty state copy |

Accent (`clay`) reserved for: focus outline, segmented tab active indicator, chart trend line, positive delta arrow color reference in `StatCard`. Do NOT use clay on standard body text, table rows, or passive labels.

**Source:** tailwind.config.ts colors block, main.css `:focus-visible`, StatCard.vue delta class logic

## Interaction Contract

| Interaction | Behaviour |
|-------------|-----------|
| Time period tab switch | Immediate client-side filter; refetch API with new period parameter; show skeleton during load |
| Stat card delta | Positive delta: green (`#4a6240`) with ↑ arrow; negative: red (`#8a4838`) with ↓ arrow; zero: `ink-faint` text, no arrow |
| Dish row hover | `rgba(184,122,78,0.04)` background tint — matches existing `.p-table tr:hover` |
| Chart hover | Tooltip showing date + count, styled: `bg-paper border border-line rounded text-sm font-mono` |
| Sidebar active state | "Analyse" entry uses `.nav-item.active` class (ink background, ink-inv text) |
| Focus visible | 2px clay outline (`#b87a4e`) at 2px offset — existing global `:focus-visible` rule |
| Keyboard nav | Tab order: time tabs → stat cards → chart → popularity table |
| Reduced motion | Chart animations disabled via existing `prefers-reduced-motion` CSS rule |
