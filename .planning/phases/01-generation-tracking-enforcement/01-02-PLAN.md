**app/pages/platform/dishes/[id].vue** -- Fetch usage data and pass `monthlyLimitReached` to GenerationStatus.

1. Add the UsageData interface and fetch near the other data fetches (after the sourceImages fetch around line 319):
```typescript
interface UsageData {
  used: number
  limit: number | null
  tierName: string
  cycleStart: string
  unlimited: boolean
}
const { data: usageData } = useLazyFetch<UsageData>('/api/user/usage', { headers: ssrHeaders })
```

2. Add a computed property:
```typescript
const monthlyLimitReached = computed(() => {
  if (!usageData.value || usageData.value.unlimited) return false
  return usageData.value.used >= (usageData.value.limit ?? Infinity)
})
```

  :image-count="sourceImages.length"
  :latest-job="latestJob"
  :monthly-limit-reached="monthlyLimitReached"
  @job-created="handleJobCreated"
/>
